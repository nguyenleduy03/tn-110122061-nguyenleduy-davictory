=== TASK ACHIEVEMENT — ACADEMIC TASK 1 ===

This criterion assesses how well the candidate describes visual data. The focus is on objective reporting, NOT opinion or explanation.

Key factors:
- OVERVIEW: Clear summary of main trends/features/comparisons
- KEY FEATURES: Most important data points selected and highlighted
- DATA SUPPORT: Comparisons and trends supported with accurate figures
- OBJECTIVITY: Describes data without giving opinions or explaining causes

Penalty factors (reduce or cap when present):
- No clear overview presented
- Describes all data mechanically without selecting key features
- Gives personal opinion or explains reasons behind trends
- Focuses on irrelevant details
- Data inaccurately reported or missing

Band ceilings:
- No overview or purely mechanical description → max TA 5.0
- Overview unclear or key features not highlighted → max TA 6.0
- Overview present but could be more comprehensive → max TA 7.0
- All requirements covered with clear effective overview → max TA 8.0
- Only TA 9 when fully satisfied with integrated overview and precise data

=== COHERENCE & COHESION PENALTY RULES ===

Penalty factors:
- Over-use of cohesive devices
- Every sentence starts with Firstly, Secondly, Moreover, Furthermore, etc.
- Mechanical linking
- Abrupt topic shifts between sentences
- Paragraphs without clear topic sentences

Band ceilings:
- Mechanical cohesion → max CC 6.0
- Reasonable cohesion but still noticeable → max CC 7.0
- Natural cohesion but not yet skilful → max CC 8.0

=== LEXICAL RESOURCE PENALTY RULES ===

Assess four dimensions: Range, Precision, Collocation, Paraphrasing.

Penalty factors:
- Key words repeated excessively
- Same vocabulary structure repeated
- Mostly B1 level vocabulary
- Academic words used unnaturally
- Incorrect collocations

Band ceilings:
- Mostly basic vocabulary → max LR 6.0
- Some less common words but used inflexibly → max LR 7.0
- Wide range but occasionally unnatural → max LR 8.0

Real vocabulary calibration for data description:
- Band 6: increased, decreased, went up, went down, big, small
- Band 7: rose, declined, peaked at, reached a low, fluctuated, significant
- Band 8: surged, plummeted, levelled off, remained stable, substantial, marginal
- Band 9: sophisticated, natural, highly precise vocabulary throughout

Describing a chart is NOT listing numbers. The writer should group data, make comparisons, and highlight patterns.

=== GRAMMATICAL RANGE & ACCURACY PENALTY RULES ===

Assess: Sentence variety, Complex structures, Accuracy, Flexibility.

Penalty factors:
- Too many simple sentences
- Only uses S + V + O patterns
- Same sentence type repeated
- Article errors
- Preposition errors
- Subject-verb agreement errors
- Punctuation errors

Band ceilings:
- Mostly simple sentences → max GRA 6.0
- Some complex sentences but limited variety → max GRA 7.0
- Varied structures but still minor errors → max GRA 8.0
- Band 9 only when: full range, near-native flexibility, only rare slips

=== REAL IELTS CALIBRATION ===

The following factors do NOT automatically lift an essay to Band 7:
- Has an introduction paragraph
- Has a conclusion sentence
- Has numbers/figures
- Has no major grammar errors

An essay only reaches Band 7+ when:
- Overview clearly highlights main trends with data
- Data is well-organised with effective comparisons
- Vocabulary and grammar clearly exceed intermediate level

=== COMMON SCORE CEILINGS ===

If vocabulary is mostly B1 level → overall max 6.5
If mostly simple sentences → GRA max 6.0
If cohesion is mechanical → CC max 6.5
If no overview → overall max 6.0
